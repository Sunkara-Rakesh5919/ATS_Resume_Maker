# ATS Resume Builder

A web-based resume builder designed to help users create Applicant Tracking System (ATS) friendly resumes.
Repository: [https://github.com/Sunkara-Rakesh5919/ATS_Resume_Maker](https://github.com/Sunkara-Rakesh5919/ATS_Resume_Maker)

## Features

-   **Resume Editor**: Easy-to-use form to input your details.
-   **Live Preview**: See your resume update in real-time.
-   **Templates**: choose from 10+ distinct resume templates.
-   **Dark Mode**: Toggle between light and dark themes.
-   **Download**: Export your resume as PDF or DOCX.
-   **Data Persistence**: Your progress is automatically saved to your browser's local storage.

## Technologies Used

-   HTML5
-   CSS3 (Custom properties for theming)
-   JavaScript (ES6+)
-   Libraries:
    -   `jspdf` (PDF generation)
    -   `html2pdf.js` (HTML to PDF conversion)
    -   `html-docx-js` (HTML to DOCX conversion)
    -   Font Awesome (Icons)

## How to Use

1.  Open `index.html` in your web browser.
2.  Fill in your details in the editor.
3.  Click "Templates" to view and select different styles.
4.  Download your resume in your preferred format.
